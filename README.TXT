It's a cheap (or whatever price) flight scanner. 

First of all, this program get the data from google sheets (destination, dates, max_price, etc). You can set many destinations, dates of flights, number of passengers, as you want
Then, the software search for flights with the given parameters.
If something below the given max prive in the given dates for the given destination(s) is found, the program sends an email.


If you fork it, you will need to get your own api endpoints and passwords.
  Resources: 
    Google Sheet Data Management - https://sheety.co/
    Kiwi Partners Flight Search API (Free Signup, Credit Card not required) - https://partners.kiwi.com/
    Tequila Flight Search API Documentation - https://tequila.kiwi.com/portal/docs/tequila_api
